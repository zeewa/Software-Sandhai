Note : To Upload The Product TSV File

  1. 1. Click on the Sample images TSV file to download the zip
  2. 2. Extract the zip file.
  3. 3. There are two tsv files in the zip archive.
  4. 4. Open the Categoryids.tsv in a spreadsheet application (Ex: Excel) to get the Category reference. ie to which Category I need to upload the Products.
  5. 5. Open the Product.tsv in a spreadsheet application
  6. 6. Don't Modify Or Delete The First Line Of Product TSV File
  7. 7. Modify The Dummy Product Details From The Second Line Of Product TSV File
  8. 8. First enter the Sub Category ID In the First Column . Refer the Categoryids TSV File
  9. 9. Then Append the Product Details In the Separate Columns
  10. 10. Finally upload the Product.tsv file with data
